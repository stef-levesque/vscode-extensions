# vscode-extensions

Various extensions for Visual Studio Code

## Installation

Simply copy the files to your vscode **extensions** folder
* **Windows** `%USERPROFILE%\.vscode\extensions`
* **Mac** `$HOME/.vscode/extensions`
* **Linux** `$HOME/.vscode/extensions`
